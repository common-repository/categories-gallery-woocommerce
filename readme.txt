Show all woocommerce product categories in gallery format.

== Description ==
This simple plugin lets you show all product categories in gallery format.
Use shortcode: "[woocategoriesgallery]" in your post or page.

If any issues regarding this plugin mail me @: sandeep.tete2013@gmail.com


== Installation ==
- Upload zip file.
- Activate the plugin.
- Scroll down to 'categories gallery Woocommerce'.
- Modify the settings.
- Done.
